# 荒岛生存游戏 (Survival Island Game)

一个基于Unity引擎开发的3D开放世界生存游戏，玩家像鲁滨逊一样在荒岛上生存，探索岛屿、建造基地、对抗敌人，并发现外星文明的秘密。

## 🎮 游戏特色

### 核心玩法
- **生存机制**：管理生命值、饥饿值、口渴值、体温
- **建造系统**：模块化建造，从简陋庇护所到坚固堡垒
- **制作系统**：收集资源，制作工具、武器、装备
- **探索系统**：探索六大生物群落，发现隐藏区域
- **战斗系统**：对抗野生动物、超自然生物、外星守卫

### 科技树系统
- **四大分支**：生存、建造、探索、外星科技
- **技能协同**：不同技能组合产生额外效果
- **进度解锁**：通过探索和研究解锁新科技

### 环境互动
- **天气系统**：动态天气影响游戏玩法
- **昼夜循环**：不同时间有不同的挑战
- **季节变化**：季节影响资源和生存条件
- **建筑维护**：建筑会受天气和生物影响

## 🛠️ 技术栈

- **游戏引擎**：Unity 2022.3 LTS
- **编程语言**：C#
- **UI系统**：UGUI + TextMeshPro
- **导航系统**：NavMesh
- **AI系统**：Behavior Tree
- **网络系统**：Netcode for GameObjects

## 🚀 快速开始

### 环境要求
- Unity 2022.3 LTS 或更高版本
- Visual Studio 2022 或 Rider
- .NET 6.0 或更高版本

### 安装步骤
1. 克隆或下载项目
2. 使用Unity Hub打开项目
3. 等待Unity导入资源
4. 打开主场景 `Assets/Scenes/MainMenu.unity`
5. 点击Play开始游戏

### 开发环境设置
1. 安装Unity 2022.3 LTS
2. 安装必要的模块（Windows Build Support等）
3. 配置Visual Studio或Rider
4. 安装推荐的插件
