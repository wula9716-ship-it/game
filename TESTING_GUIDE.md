## 🧪 测试步骤

### 第一阶段：基础功能测试

#### 1. 创建测试场景
```
1. 在 Unity 中创建新场景：File > New Scene
2. 保存场景：File > Save As > "TestScene"
3. 添加地面：
   - 创建 Plane：GameObject > 3D Object > Plane
   - 设置位置：(0, 0, 0)
   - 设置缩放：(10, 1, 10)
   - 设置 Layer 为 "Ground"
```

#### 2. 设置玩家角色
```
1. 创建空物体：GameObject > Create Empty
2. 重命名为 "Player"
3. 添加组件：
   - Character Controller
   - Player Controller 脚本
   - Player Stats 脚本
   - Inventory System 脚本
4. 添加子物体 Camera
5. 设置 Tag 为 "Player"
6. 设置 Layer 为 "Player"
```

#### 3. 测试玩家控制
```
测试项目：
☐ WASD 移动
☐ 鼠标视角转动
☐ 跳跃（Space）
☐ 跑步（Shift）
☐ 交互（E）

预期结果：
- 移动流畅，无卡顿
- 视角转动平滑

- 交互有射线检测
```

## 🐛 常见问题排查

### 问题1：脚本编译错误
```
症状：Console 显示红色错误
解决：
1. 检查所有脚本是否有语法错误
2. 确保所有引用的类和方法存在
3. 重新导入脚本：Assets > Reimport All
```

### 问题2：玩家无法移动
```
症状：按键无反应
解决：
1. 检查 PlayerController 脚本是否挂载
2. 检查 Character Controller 组件
3. 检查输入设置：Edit > Project Settings > Input
```

### 问题3：UI 不显示
```
症状：HUD 或界面不出现
解决：
1. 检查 Canvas 是否存在
2. 检查 UI 脚本是否挂载
3. 检查 EventSystem 是否存在
```

### 问题4：物品无法使用
```
症状：点击使用无效果
解决：
1. 检查 ItemData 是否正确设置
2. 检查 isEdible/isUsable 标记
3. 检查使用逻辑代码
```
